# zigLocks

⚠️ **This package is still under development and may have breaking changes.**

A Zig library for thread synchronization primitives and locking mechanisms.

## Structure

- `src/main.zig` - Main entry point
- `src/locks/` - Lock implementations and utilities
- `build.zig` - Build configuration
- `build.zig.zon` - Package configuration

## Building

```bash
zig build
```

## Usage

This library provides various locking primitives for concurrent programming in Zig.
